# 🗂️ git_control — SOOJUNBARNG リポジトリ管理

全リポジトリの一覧・管理スクリプトをまとめたハブリポジトリ。

## 使い方

```bash
# 全リポジトリの状態確認
./scripts/status.sh

# 全リポジトリを最新に更新（pull）
./scripts/pull_all.sh

# リポジトリ一覧を表示
./scripts/list.sh
```
